# Varnotdefteri
Sürekli plan yapan birisiyim ama sürekli unutuyorum ve düzenlemek için kendime veri toplamayan bir not defteri yapmak istedim belki işinize yarar

Özellikler
--
-Not ekleme ve not silme

Kurulum
--

Önkoşullar

    Visual Studio Community (Tercihen 2026 sürümü)
    Uygulamanın kullandığı sürüm: [.NET 8.0 SDK] 

Adımlar

    -Projeyi Klonlayın:
    Bash
    git clone https://github.com/CigkofteXL/Varnotdefteri.git
    -Visual Studio'da Açın:
        NotDefteri klasörünün içindeki NotDefteri.sln çözüm dosyasını (Solution File) çift tıklayarak Visual Studio'da açın.
    -Bağımlılıkları Yükleyin:
        Gerekiyorsa, çözüm dosyasını açtıktan sonra NuGet paketlerinin otomatik olarak geri yüklenmesini bekleyin. (Genellikle VS bunu otomatik yapar.)
    -Uygulamayı Derleyin:
        Üst menüden Build (Derleme) > Build Solution (Çözümü Derle) seçeneğini seçin.
    -Uygulamayı Başlatın:
        Araç çubuğunda Start (Başlat) (Yeşil Üçgen) butonuna basın veya klavyeden <kbd>F5</kbd> tuşuna basarak uygulamayı çalıştırın (Debug modunda).
Bu adımlar, projeyi indirip çalıştırmak için gereken en temel ve evrensel yöntemlerdir.
